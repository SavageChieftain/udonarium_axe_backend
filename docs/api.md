# API リファレンス

## `GET /`

ウェルカムエンドポイント。

**レスポンス:**

```json
{
    "message": "Ready to serve your realm."
}
```

## `GET /v1/status`

ヘルスチェック用エンドポイント。

**レスポンス:**

```
200 OK
```

## `POST /v1/skyway2023/token`

SkyWay 2023 の JWT トークンを発行します。

**リクエスト:**

```json
{
    "formatVersion": 1,
    "channelName": "room-name",
    "peerId": "peer-id"
}
```

| フィールド      | 型       | 必須 | 説明                                |
| --------------- | -------- | ---- | ----------------------------------- |
| `formatVersion` | `int`    | ◯    | フォーマットバージョン（`1`）       |
| `channelName`   | `string` | ◯    | ルームチャンネル名（最大 200 文字） |
| `peerId`        | `string` | ◯    | ピア ID（最大 200 文字）            |

**レスポンス:**

```json
{
    "token": "eyJ..."
}
```

## エラーレスポンス

すべてのエラーは JSON 形式で返されます。

```json
{
    "error": "エラーメッセージ"
}
```

| ステータス | 条件                                             |
| ---------- | ------------------------------------------------ |
| `400`      | Origin ヘッダーなし、または リクエストボディ不正 |
| `403`      | 許可されていないオリジンからのリクエスト         |
| `404`      | 存在しないエンドポイント                         |
| `405`      | 許可されていない HTTP メソッド                   |
| `413`      | リクエストボディが 64KB を超過                   |
| `415`      | Content-Type が `application/json` 以外          |
| `500`      | サーバー内部エラー（設定の読み込み失敗等）       |
